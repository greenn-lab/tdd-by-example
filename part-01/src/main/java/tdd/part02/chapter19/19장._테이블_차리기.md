# 테이블 차리기

테스트를 작성하다 보면 공통된 패턴을 발견한다는데,
 1. 준비(arrange) - 객체를 생성하고
 2. 행동(act) - 어떤 자극을 주고
 3. 확인(assert) - 결과를 검사
 
아직 확~ 와닿는 얘긴 아닌것 같네요.
